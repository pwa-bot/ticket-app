## Implementing

Anyone can implement this protocol. The specification defines:

- Directory structure (`.tickets/tickets/`, `.tickets/index.json`)
- Ticket file format (Markdown with YAML frontmatter)
- Required and optional fields
- State machine and transitions
- Actor attribution format

See `PROTOCOL.md` for details.
